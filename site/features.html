A full suite of moderation tools for managing your community.

/kickKick members instantly
/muteTimed mutes with duration
/timeoutDiscord native timeout
/warnWarning system with history
/purgeBulk message deletion
/lockdownChannel lock/unlock
/softbanBan + unban to purge msgs

Moderation History

Every moderation action is logged and can be reviewed with /hist <user>. Warnings can be viewed with /warns [user] and removed with /unwarn.

Punishment DM Notifications

When enabled via /punishment-dm, users receive a DM notification when they are warned, muted, banned, or kicked — including the reason and duration. Users can also appeal punishments through the built-in appeal system.

Economy & Banking

A full virtual currency system with earning, spending, gambling, and banking.

Earning

  • /daily — Claim daily rewards (24h cooldown)
  • /work — Work a random job to earn points
  • /rob <user> — Risk robbing another user for their points

Spending & Trading

  • /pay <user> <amount> — Transfer points to another user
  • /chargeback <id> <reason> — Reverse a transaction (admin)

Banking

  • /bank deposit <amount> — Deposit into your bank (protected from robbery)
  • /bank withdraw <amount> — Withdraw from your bank
  • /bank balance — View bank and wallet balances
  • Loan system with interest and repayment tracking

Leveling System

Members earn XP by chatting. XP accumulates and unlocks levels with configurable role rewards.

  • Automatic XP — users earn XP for sending messages (with cooldowns to prevent spam).
  • Level-based roles — automatically assign roles when users reach specific levels.
  • /rank [user] — view your level, XP, and progress to the next level.
  • /leaderboard / /lb — see the server's XP leaderboard.
  • Admin controls — /xp and /level commands to add, subtract, or set values.
Enable/disable the leveling system with /levels enable/disable. Configure level roles through the settings.

Proxy System

A PluralKit-style proxy system that lets users send messages as different identities using webhooks.

How It Works

  1. Create a proxy member with /proxy create <name>
  2. Set proxy tags (prefix/suffix) — e.g., text so typing {Hello!} sends as that member
  3. Set a custom avatar with /proxy avatar <name> <url>
  4. Messages matching your tags are automatically proxied via webhooks

Features

  • Multiple members — create as many proxy identities as you need
  • Groups — organize members into groups
  • Custom avatars — each member can have its own avatar
  • Guild-specific or global — proxies can work in one server or all
  • Switch system — /proxysettings switch <member> to auto-proxy all messages
  • Proxy indicator — servers can enable [proxied] tag for transparency

Cross-Server Global Chat

Link channels across multiple servers so messages relay in real-time.

How It Works

  1. Create a global chat channel with /globalchat create <name>
  2. Link servers by having admins join with /globalchat join <code>
  3. Messages sent in linked channels are relayed to all other linked channels via webhooks

Safety Features

  • Attribution — relayed messages are prefixed with [GC] in the webhook username for clear distinction from real users
  • Rate limiting — 3-second per-user cooldown to prevent spam
  • Content limits — messages capped at 2000 characters, max 5 attachments
  • Server muting — mute specific servers from sending or receiving
  • Moderation — global chat owners can kick/ban servers
  • Rules — set channel-specific rules viewable by all linked servers

Ticket System

A support ticket system for managing user inquiries and issues.

  • Ticket panels — create embed panels with a button to open tickets
  • Auto-creation — clicking the button creates a private channel for the user and staff
  • Close & archive — tickets can be closed, archived, or deleted
  • Transcript logging — ticket conversations can be logged for review

Auto-Moderation

Automated message filtering to keep your server clean.

  • Spam detection — configurable thresholds for rapid message sending
  • Link filtering — block or whitelist specific domains
  • Mention spam — limit mass mentions
  • Caps filter — detect and act on excessive caps
  • Invite blocking — automatically remove Discord invite links
  • Word blacklist/whitelist — custom word filtering
  • Bypass permission — users with automod.bypass are exempt

Configure with /automod config and /antispam.

Comprehensive Logging

Log server events to designated channels for transparency and record-keeping.

  • Message logs — edits, deletions, and bulk deletes
  • Moderation logs — bans, kicks, mutes, warns, and all mod actions
  • Join/leave logs — member join, leave, and role changes
  • Auto-logging — automatically detect and log events
  • Manual logs — /log for staff to create manual entries
  • Selective log types — /logchannel to configure which events go to which channels

Welcome System

  • Welcome messages — customizable messages when members join
  • Auto-roles — automatically assign roles to new members
  • Configurable channel — choose which channel receives welcome messages
  • Variable support — use placeholders for user names, server names, member counts, etc.

Configure with /welcome.

Suspicious Account Detection

Automated system that analyzes new members and flags potentially suspicious accounts.

  • Account age analysis — flags very new accounts
  • Pattern detection — identifies bot-like username patterns
  • Risk scoring — multi-factor suspicion level calculation
  • Admin notifications — alerts sent to designated channels or DMs
  • Validate/invalidate — staff can mark accounts as safe or suspicious
  • Global masterlist — bot-owner managed cross-server suspicious user tracking

Configure notifications with /suspiciousnotify. Manage the masterlist with /suspiciouslist.

Role Persistence

When a member leaves and rejoins the server, their roles can be automatically restored.

  • Configurable expiry period — roles are stored for a set duration
  • Enable/disable with /rolepersistence

Reaction Roles

Let members self-assign roles by reacting to messages.

  • Create reaction role panels with /reactionrole
  • Supports custom and Unicode emojis
  • Add or remove roles by clicking reactions

Pride & Inclusivity

  • /pride <flag> — apply pride flag overlays to your avatar
  • /flags — list all available pride flags
  • /pronouns — set pronoun roles for yourself

Server Backups

Create and restore snapshots of your server configuration with /backup create and /backup restore. Backups include settings, permissions, logging config, and more.

Privacy & Data

  • /privacy — view the bot's privacy policy directly in Discord
  • /deletedata — permanently delete all your personal data with confirmation
